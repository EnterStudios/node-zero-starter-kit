#Node JS Zero - Starter Kit

##About
Node Zero starter kit is a starting point for web application with Node JS and Angular JS2. It bases on:

    * Node JS
    * Express JS
    * ES6
    * Gulp
    * Passport
    * Mocha, supertest
    * AngularJS 2

## Getting Started

#### Install gulp globally
```sh
$ npm install --global gulp-cli
```
#### Install dependency
```sh
$ npm install
```
#### Run
* Development mode:
```sh
$ gulp
```
* unit test:
```sh
$ gulp test
```
* Release mode:
```sh
$ gulp build:release
$ npm start
```
